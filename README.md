# Trabalho_1_Machine_Learning

Análise multivariada de demanda energética horária na Espanha utilizando Machine Learning. O projeto explora a correlação entre variáveis meteorológicas de 5 grandes cidades e o consumo elétrico nacional por meio dos conjuntos de dados diponibilizados em https://www.kaggle.com/datasets/nicholasjhana/energy-consumption-generation-prices-and-weather.
